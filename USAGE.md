# CRM МБЗ На Netlify

После публикации сайт открывается с корня и автоматически переходит в CRM из папки `app/`.

## Быстрая Проверка

1. Откройте опубликованную Netlify-ссылку.
2. Проверьте логотип МБЗ.
3. Проверьте разделы `Панель ГД`, `Первые 5 заказов`, `Клиенты ГНБ`, `Объекты ГНБ`, `Образцы`.
4. Откройте ссылку на телефоне.

## Домен

Для домена `crm.mbz-bentonit.ru` добавьте в DNS запись:

```text
Тип: CNAME
Host: crm
Target: crm-mbz.netlify.app
```

Подробная инструкция: `docs/netlify-publishing.md`.
