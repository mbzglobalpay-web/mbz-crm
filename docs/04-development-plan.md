# План разработки CRM МБЗ

## Этап 0. Архитектура и прототип

- Утвердить ТЗ.
- Утвердить модель данных.
- Утвердить карту интерфейсов.
- Собрать интерактивный прототип с доменными сущностями рынка ГНБ.
- Проверить сценарии с коммерческим отделом, технологами и логистикой.

## Этап 1. Backend MVP

- PostgreSQL schema и миграции.
- API для компаний, контактов, сделок, действий, КП, образцов, испытаний, заказов, поставщиков и документов.
- RBAC и аудит.
- Импорт текущей клиентской базы.
- Поиск и дедупликация.

## Этап 2. Frontend MVP

- Рабочие реестры.
- Карточка компании.
- Воронка сделок.
- Очередь задач.
- Создание КП и заказов.
- Документы и статусы согласования.

## Этап 3. Интеграции

- Телефония.
- Email.
- 1С/ERP.
- ЭДО.
- Dadata/ФНС.
- Транспортные компании.
- BI-выгрузки.

## Этап 4. Производственный контур

- Резервное копирование.
- Мониторинг.
- Ролевая модель.
- Журнал доступа.
- Регламенты работы менеджеров и лаборатории.
- Обучение пользователей.

## Этап 5. Оптимизация

- Скоринг клиентов и сделок.
- Прогноз повторных заказов.
- Аналитика эффективности образцов.
- Контроль маржинальности.
- Автоматические рекомендации по следующему действию.
