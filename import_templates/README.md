# Шаблоны импорта реальных данных МБЗ

Папка содержит шаблоны для подготовки CRM МБЗ к загрузке реальных данных по рынку ГНБ.

## Файлы

- `xlsx/mbz_hdd_import_templates.xlsx` - единая Excel-книга со всеми шаблонами.
- `csv/01_clients_hdd_template.csv` - импорт клиентов ГНБ.
- `csv/02_decision_makers_template.csv` - импорт ЛПР.
- `csv/03_hdd_objects_template.csv` - импорт объектов ГНБ.
- `csv/04_samples_template.csv` - импорт образцов.
- `csv/05_orders_template.csv` - импорт заказов.
- `csv/06_yfo_first_100_companies_structure.csv` - структура для первых 100 компаний ГНБ ЮФО.

## Порядок загрузки

1. Заполнить клиентов ГНБ.
2. Заполнить ЛПР и связать их с клиентами через `client_id`.
3. Заполнить объекты ГНБ и связать их с клиентами через `client_id`.
4. Заполнить образцы и связать их с клиентами и объектами через `client_id` и `object_id`.
5. Заполнить заказы и связать их с клиентами, объектами, образцами, испытаниями и КП.

## Правила ID

- `client_id` - главный ключ клиента. Для ЮФО используйте формат `C-YFO-001`.
- `contact_id` - ключ ЛПР. Рекомендуемый формат `P-YFO-001`.
- `object_id` - ключ объекта ГНБ. Рекомендуемый формат `G-YFO-001`.
- `sample_id` - ключ образца. Рекомендуемый формат `S-YFO-001`.
- `order_id` - ключ заказа. Рекомендуемый формат `O-YFO-001`.

## Проверка перед загрузкой

- Нет пустых `client_id`, `object_id`, `sample_id`, `order_id`.
- Нет дублей ИНН.
- Все `client_id` в ЛПР, объектах, образцах и заказах существуют в листе клиентов.
- Все `object_id` в образцах и заказах существуют в листе объектов.
- Даты заполнены в формате `YYYY-MM-DD`.
- Вероятность заказа указана числом от 0 до 100.
- Потенциал и тоннаж указаны числами без единиц измерения.
